## 주의사항
이 코드는 제가 수행한 첫 번째 프로젝트의 프론트엔드 부분에 추가한 것입니다. 포트폴리오 배포를 위해 개발했으며 리액트에 대한 기본적인 지식을 바탕으로 기능을 추가하고 fetch 문을 수정하였습니다.  

기능 추가 부분에서는 기존의 productList에서는 categoryId를 필터링하는 기능이 없었지만, 제가 코드를 추가하여 이를 가능하게 만들었습니다. 또한, 서버에서 데이터를 가져오는 fetch 부분도 상황에 따라 수정하여 추가하였습니다.  

저는 주로 백엔드 개발에 초점을 맞추고 있어서 프론트엔드 부분에 대한 이해가 아직 완벽하지 않습니다. 이 점 양해해 주시기 바랍니다.
